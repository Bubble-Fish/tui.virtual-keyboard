    VirtualKeyboard

    가상키보드
    실제 키보드가아닌 가상의 키보드를 만들어, 숫자, 영문, 한글, 특수문자 등의 키 입력 효과를 제공합니다.

    Feature

    • 가상의 키보드를 제공한다.
    • 한영, 숫자, 특수문자 변환이 가능하다.
    • 가상키보드를 열고 닫을때, 키보드의 판을 변경할때 사용자가 콜백을 등록하여 사용할수 있다.
    • 셔플을 제공한다.
    • 모바일, 데스크탑 환경을 모두 지원한다.
